Each example in this section is the end product of a series of exercises in the course. The examples are all simple but each teaches a point about the usage of D3. Mike Bostock himself is the foremost proponent of the example driven approach to learning D3. He has included numerous examples of D3 visualizations on d3js.org and has created the site bl.ocks.org as a repository for others to share their D3 visualizations with the D3 community. It is striking to see how little D3 code needs to be written to support very sophisticated visualizations. This is a testimony to the power of the D3 API.
