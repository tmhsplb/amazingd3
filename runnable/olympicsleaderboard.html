Again, we can certainly render this visualization without D3. But this is not the point of this example. Notice that the title says that this is the leaderboard for the Feb7 dataset. The point is that there is also a Feb8 dataset that can be loaded to update the leaderboard without a page transition. The Feb8 dataset updates medal counts for all teams. The leaderboard is designed to always list the top 3 teams in the olympic standings. The Feb8 dataset implies that one of the teams is replaced by a new team and that the rows are reordered to reflect a new leader, team USA.

This is possibly the most difficult example in the course. It teaches about the full enter-update-exit pipeline that is an innovation of D3. The course will describe a pattern to use to manage the complexity of this pipeline.
