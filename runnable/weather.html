This example, due to Jan Milosh, is a custom visualization; it cannot be produced by any commercial charting package. The example demonstrates the ability of D3 to connect to a remote data source, in this case the Weather Underground API provided by the Weather Channel. By default the weather for Houston, Texas is displayed. The API provides an autocomplete input that allows you to query for current data for cities around the world.
